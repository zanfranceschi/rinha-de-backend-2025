## Techs Utilizadas

- **.NET 9** (com Minimal API)
- **.NET Channels** para fila
- **Background Workers** para processamento
- **Docker** para conteinerização e orquestração

## Requisitos

- .NET 9
- Docker e Docker Compose

## Como executar

1. Clone o repositório
2. Execute `docker-compose up --build`
